Sensoren

Jeden Sensor den du verwenden möchtest aktivieren und konfigurieren.

HX711 — Waage (Load Cell)
DS18B20 — Temperatur (1-Wire)
BME280 — Temp / Feuchte / Luftdruck (I2C)
BME680 — Temp / Feuchte / Luftdruck / Gas (I2C)
DHT22 / DHT11 — Temp / Feuchte (GPIO)
SHT31 / SHT25 — Temp / Feuchte präzise (I2C)
AHT10 / AHT20 — Temp / Feuchte (I2C)
BH1750 — Licht / Beleuchtungsstärke (I2C)
ADS1115 — ADC / Batteriespannung (I2C)
Spannungsteiler: z.B. 5.7 für 100kΩ/20kΩ Teiler (12V Akku → 3.3V Eingang)
